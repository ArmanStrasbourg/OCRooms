Dépot distant
***************************
3 zones locales majeurs: 
- le répertoire de travail (working directory / WD)
- l'index, ou stage 
- le dépôt local (Git directory / repository)
WD --> /via Git Add/ --> Stage 
                              --> /via Git Commit/ --> Repository
             | --> /via Git Push/ --> | 
Repository   | <-- /via Git Pull/ <-- | GitHub


Initialiser le dépot :
git init

git remote add OC https://https://github.com/OpenClassrooms-Student-Center/ProjetOpenSource.git

git clone https://github.com/OpenClassrooms-Student-Center/ProjetOpenSource.git
